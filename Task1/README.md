# Task 1 - Basic Network Sniffer

## Objective

Develop a Python-based network packet sniffer to capture and analyze network traffic using Scapy.

## Description

This project captures network packets and displays useful information such as source IP address, destination IP address, protocol type, and port details. It provides a basic understanding of how data travels across a network.

## Features

* Captures network packets
* Displays source and destination IP addresses
* Identifies TCP, UDP, and ICMP protocols
* Displays source and destination port numbers
* Helps understand basic network communication

## Technologies Used

* Python
* Scapy

## Learning Outcomes

* Understanding network packets
* Basic network traffic analysis
* Familiarity with common network protocols
