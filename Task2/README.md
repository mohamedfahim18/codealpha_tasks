# Task 2 - Phishing Awareness Training

## Objective

Create an awareness program to educate users about phishing attacks and safe online practices.

## Description

This task includes a phishing awareness presentation along with an interactive Python quiz. It explains common phishing techniques, social engineering attacks, fake websites, and methods to identify and prevent phishing attempts.

## Contents

* Phishing Awareness Presentation
* Interactive Phishing Quiz

## Topics Covered

* Introduction to Phishing
* Types of Phishing Attacks
* Fake Websites
* Social Engineering
* Prevention Techniques
* Cybersecurity Best Practices

## Technologies Used

* Microsoft PowerPoint
* Python

## Learning Outcomes

* Understanding phishing attacks
* Recognizing suspicious emails and websites
* Applying safe online security practices
