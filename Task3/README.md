# Task 3 - Secure Coding Review

## Objective

Review a Python login application to identify security vulnerabilities and recommend secure coding practices.

## Description

This task involves manually reviewing a simple Python login program to identify common security weaknesses and document recommendations for improving application security.

## Review Method

* Manual Code Inspection

## Vulnerabilities Identified

* Hardcoded credentials
* Plain-text password verification
* No brute-force protection
* No multi-factor authentication

## Recommendations

* Use password hashing
* Store credentials securely
* Implement account lockout after multiple failed login attempts
* Enable multi-factor authentication

## Learning Outcomes

* Identifying common security vulnerabilities
* Understanding secure coding principles
* Learning practical remediation techniques
